# What Is "CrochetCraft.org"

crochetcraft.org is a free website, mobile app, & desktop application that is made for crafters & crocheters alike!

# What features are part of this project?

The features in the project are updated frequently, HOWEVER here are some of the basic features to the project!!

  1. Materials tracking
  2. Project tracking
  3. Patterns tracking (avaliable for crochet, knitting, & sewing only!)
  4. Login/Signup screen
  5. Optional email list
  6. Finished projects tracker
  7. & More!!

# What is the style of the app/website?

Very colorful but minimalist in a way! (just visit it so you see what i mean xD)

# What should i know about the project?

Nothing really, other than its very fun and has a fun community!
